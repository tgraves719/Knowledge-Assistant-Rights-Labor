Small uppercase pill for status and citation provenance (MOA / Base / MOA+Base tags on citation badges).

```jsx
<Badge tone="citation">MOA</Badge>
<Badge tone="success">Recovered</Badge>
<Badge tone="warning">Needs Review</Badge>
```

Tones: `neutral`, `citation` (gold, matches product's `.citation-badge`), `success`, `warning`, `danger`, `info`.
